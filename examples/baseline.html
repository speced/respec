Baseline Badge Example

This example demonstrates the Baseline browser availability badge. It auto-detects from the edDraftURI or you can set implementationStatus: "feature-id" explicitly.

This is an example document to test the baseline badge feature.

Configuration Options

The implementationStatus config option accepts:

  • true — auto-detect from spec URLs
  • "feature-id" — explicit web-features ID
  • { feature: "id", removeOnSave: true } — full options
